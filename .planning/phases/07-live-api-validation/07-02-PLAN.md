Create `scripts/swagger-coverage.mjs` -- a standalone script that produces a readable text report comparing the PriceLabs Swagger API endpoints against implemented MCP tools.

**Approach:** This is a static analysis script (no API calls needed). It compares two hardcoded lists:

**PriceLabs Swagger API endpoints** (from https://app.swaggerhub.com/apis-docs/Customer_API/customer_api/1.0.0-oas3):
```
GET  /v1/listings                   - Get all listings
GET  /v1/listings/{id}              - Get single listing
POST /v1/listings                   - Update listings
POST /v1/listing_prices             - Get prices for a listing
GET  /v1/neighborhood_data          - Get neighborhood comparison data
GET  /v1/reservation_data           - Get reservations
GET  /v1/fetch_rate_plans           - Get rate plans
POST /v1/push_prices                - Push prices to PMS
GET  /v1/overrides                  - Get price overrides
POST /v1/overrides                  - Set price overrides
DELETE /v1/overrides                - Delete price overrides
POST /v1/sync_listings              - Trigger listing sync
```

```
pricelabs_get_listings         -> GET  /v1/listings

pricelabs_get_overrides        -> GET  /v1/overrides
pricelabs_set_overrides        -> POST /v1/overrides
pricelabs_delete_overrides     -> DELETE /v1/overrides
pricelabs_sync_listings        -> POST /v1/sync_listings
```

**Plus internal tools (no API endpoint):**
