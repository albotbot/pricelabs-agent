```json

  "name": "Prism",

          "PRICELABS_WRITES_ENABLED": "false"

      }
    ]
  }

Before writing the file back:

6. Verify `gateway.tailscale.mode` is still `"off"` -- ABORT if changed

**Step 5: Write the modified config**
Write the validated JSON back to ~/.openclaw/openclaw.json with proper formatting (2-space indent).

```
IMPORTANT: Use `sudo systemctl restart openclaw-gateway.service` (system-level). NEVER use `systemctl --user`. Always back up before running `openclaw doctor --fix`.
